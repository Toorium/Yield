## What is Yield?
Yield is a beginner-friendly programming language that runs fast,
requires zero setup, and was designed to be the easiest first language
for anyone coming from Roblox scripting or with no coding experience.

## Features
- Clean readable syntax
- Standalone — no Python or dependencies
- Classes, functions, loops, lists built in
- Terminal games with key.get()
- VS Code syntax highlighting

## Install
1. Download yield.exe from [Releases](../../releases)
2. Add C:\Yield to your PATH
3. Run yield myfile.yd

## Quick Example
